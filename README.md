#  Student Management System (Python)

A simple **menu-driven Python application** to manage student records, subject-wise marks, and academic performance.  
This project is suitable for beginners learning **Python and Object-Oriented Programming (OOP)**.

---

##  Features

- Add student details (Student ID and Name)
- Add multiple subjects for each student
- Store marks for each subject (3 marks per subject)
- Calculate:
  - Total marks
  - Average marks
  - Grade (A, B, C, D)
- Display all student records
- Easy-to-use command-line menu

---

##  Technologies Used

- Python 3
- Core Python concepts:
  - Classes & Objects
  - Dictionaries
  - Loops and Conditionals
  - Functions
  - User Input Handling
